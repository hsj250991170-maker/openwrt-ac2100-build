# 小米/红米 AC2100 OpenWrt 固件使用说明

## 固件信息
- **基础系统**：ImmortalWrt 24.10
- **目标设备**：小米AC2100 + 红米AC2100 (MT7621)
- **代理功能**：passwall2 + OpenClash + sing-box + xray-core + naiveproxy
- **主题**：Argon
- **语言**：简体中文
- **默认LAN IP**：192.168.31.1
- **默认密码**：空（首次登录后请设置）

## 固件文件说明
编译产物包含以下文件：

### 固件文件（Breed刷机用）
- `openwrt-24.10.x-ramips-mt7621-xiaomi_mi-router-ac2100-squashfs-sysupgrade.bin` — 小米AC2100 升级固件
- `openwrt-24.10.x-ramips-mt7621-xiaomi_redmi-router-ac2100-squashfs-sysupgrade.bin` — 红米AC2100 升级固件
- 同名 `factory.bin` — 从原厂固件刷入用（如果你已经刷了Breed，用 sysupgrade.bin 即可）

### 关键 ipk 包（备用，可单独安装）
- `luci-app-passwall2*.ipk` — passwall2 主界面
- `luci-app-openclash*.ipk` — OpenClash 主界面
- `xray-core*.ipk` / `sing-box*.ipk` — 代理核心
- `mihomo*.ipk`（如有）— mihomo 内核

## 刷机步骤（Breed 不死鸟）

### 1. 进入 Breed
- 路由器断电
- 用牙签/卡针按住 Reset 键不松
- 插上电源，等5-10秒（指示灯变蓝闪烁）
- 松开 Reset
- 电脑浏览器访问 http://192.168.1.1 进入 Breed

### 2. 刷入固件
- 在 Breed 界面点「固件更新」
- 「闪存布局」选择 `Xiaomi R3G OpenWrt`（小米/红米AC2100通用）
- 在「固件」栏上传对应的 sysupgrade.bin
- ⚠️ 勾选「Flash」选项中的 Flash Layout 后再点上传
- 点「上传」→ 确认 → 等待刷写完成自动重启

### 3. 首次进入 OpenWrt
- 重启后（约2-3分钟），电脑访问 http://192.168.31.1
- 用户名 `root`，密码空
- **立即设置管理密码**：系统 → 管理权 → 主机密码

## 代理配置

### passwall2（常规代理）
1. 服务 → PassWall2 → 基本设置
2. 开启 TCP 代理（建议用 GFWList 模式）
3. 节点列表 → 添加/订阅节点
4. 选择节点 → 保存并应用

### OpenClash（用于 Ninja 订阅）
1. 服务 → OpenClash → 全局设置
2. **首次使用必须先下载 mihomo 内核**：
   - 进「版本更新」标签 → 内核管理 → 选 Meta mipsel-softfloat 版 → 下载
   - 或 SSH 进路由器执行：`/usr/bin/oc-mihomo-helper.sh`
3. 配置文件订阅 → 添加你的 Ninja/clash 订阅链接
4. 启动 OpenClash

### 关于 Ninja 协议
Ninja 是基于 mihomo 修改的私有代理内核。AC2100 是 mipsel 架构，Ninja 官方 ipk 不支持。本固件方案：
- 用 OpenClash + 标准 mihomo mipsel 内核加载 clash 格式订阅
- 如果你的机场 Ninja 订阅支持 clash 格式 → 直接可用
- 如果只支持 Ninja 私有客户端 → 可能不兼容，需联系机场确认

## 常见问题

**Q: 刷完无法访问 192.168.31.1？**
A: 检查电脑IP是否在 192.168.31.x 网段；或断电按 Reset 5秒进 Breed 重新刷。

**Q: OpenClash 启动失败？**
A: 99% 是没下载 mihomo 内核。先去「版本更新」标签下载 mipsel-softfloat 内核。

**Q: passwall2 节点不工作？**
A: 检查节点配置；确认 xray-core 已正确安装；查看 passwall2 日志。

**Q: WiFi 没开？**
A: 网络 → 无线 → 启用 radio0 (2.4G) 和 radio1 (5G)。

## 恢复与备份
- **Breed 备份**：进 Breed → 固件备份 → 下载 EEPROM 和固件分区
- **OpenWrt 内备份**：系统 → 备份/升级 → 生成备份
- **恢复出厂**：系统 → 备份/升级 → 执行重置；或 Breed → 固件更新 → 勾选 EEPROM 清空
